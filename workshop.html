Enterprise CI/CD Pipeline Workshop

Workshop Overview: Build a complete enterprise-grade CI/CD pipeline for a full-stack application using AWS services, GitHub Actions, and modern DevOps practices.

Quick Deploy Options

Option 1: GitHub Pages

  1. Enable GitHub Pages in repository settings
  2. Push the deploy-docs.yml workflow
  3. Access at: https://yourusername.github.io/repository-name

Option 2: AWS S3 Static Website

# Create S3 bucket
aws s3 mb s3://your-workshop-bucket

# Upload files
aws s3 sync . s3://your-workshop-bucket --exclude "*.git/*"

# Enable static website hosting
aws s3 website s3://your-workshop-bucket --index-document workshop.html

Option 3: Local Hugo Server

# Install Hugo
winget install Hugo.Hugo.Extended

# Create site
hugo new site workshop
cd workshop
mkdir -p content/workshops/enterprise-cicd
copy ..\aws-fcj-workshop.md content\workshops\enterprise-cicd\_index.md

# Run locally
hugo server -D
Recommendation: Use GitHub Pages for public workshops or S3 for internal company documentation.
